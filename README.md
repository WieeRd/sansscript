# sansscript
## 문법 설명서

### 괄호 정리
1. `{변수}` - 변수 값
2. `(문자열)` - 문자열
3. `[변수]` - 변수의 길이
4. `<함수>` - 함수 실행 (인수랑 전역변수랑 대체)(...?)
5. `/계산식\` - 계산

### 값 
`{변수}(문자열)[변수]<함수>/계산식\`

순서는 상관 없다

` 헐 ` 은 한줄 코드로 써야할때 컴파일러에서 줄바꿈으로 인식(양끝에 띄어쓰기)

1. **와!** - 대입

사용 방법 - 
와! `{변수이름}` `값`

2. **샌즈** - 출력

사용 방법 - 
샌즈 `값`

3. **파피루스** - 입력

사용 방법 - 
파피루스 `{변수}`

4. **않임?** - 조건문

사용 방법 - 
않임? `값`가`값`면 코드

5. **머멍이** - 반복문

사용 방법 -
머멍이 `값(꼭 숫자 아니면 오류)`동안 코드

6. **토비폭스** - 함수 정의

사용 방법 - 
토비폭스 <함수이름> 코드....


### 변수 관련
`샌즈이긴횟수` - 처음에는 0, 반복문 돌때마다 1 증가


### 함수 관련
반환 정의가 없으면 `키키` 반환

샌즈더스켈레톤 `값` 은 반환문
